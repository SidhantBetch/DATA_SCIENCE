# 🏛️ Types of Variables
- Numerical variables:- These represent numbers and allow mathematical operations.
  
  - Discrete:- Countable values (usually integers). No values in between
    - Examples:
      - Number of students in a class (10, 11, 12…)
      - Number of cars
      - Goals scored in a match
        
  - Continuous:- Can take any value within a range. Measured, not counted
    - Examples:
      - Height (170.5 cm, 170.55 cm…)
      - Weight
      - Temperature
      - Time
        
- Categorical:- These represent labels or categories, not numbers.
  
  - Ordinal:- Categories have a meaningful order, but differences aren’t measurable
    - Examples:
      - Rankings (1st, 2nd, 3rd)
      - Satisfaction level (low, medium, high)
      - Education level
        
  - Nominal:- No order or ranking
    - Examples:
      - Gender
      - Color (red, blue, green)
      - Country
        
- Dates & Times:- Special type of variable dealing with time-based data
  - Examples:
    - Date of birth
    - Timestamp (2026-04-06 10:30 AM)
    - Day of the week
      
- Mixed:- ariables that combine multiple types or need special handling.
  - Examples:
     - A dataset with both numbers and categories
     - Text + numerical data
     - Structured + unstructured data
👉 In Machine Learning, these often require:
     - Encoding (for categorical data)
     - Normalization (for numerical data)
