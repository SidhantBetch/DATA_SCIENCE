# 🔥 Feature Scaling Techniques in Machine Learning

## 📌 What is Feature Scaling?

Feature Scaling is a technique used to **standardize or normalize numerical features** so that all variables contribute equally to a machine learning model.

👉 Example:
- Age → 20 to 60  
- Salary → 20,000 to 1,00,000  

➡️ Large values dominate small ones ❌  
➡️ Scaling fixes this ✅  

---

## 🧠 Why Feature Scaling is Important?

- Improves model accuracy  
- Speeds up convergence  
- Prevents bias toward large values  

### Models that require scaling:
- Linear Regression  
- Logistic Regression  
- KNN  
- SVM  

### Models that don’t require scaling:
- Decision Tree  
- Random Forest  

---

## ⚙️ Types of Feature Scaling Techniques
1. Standardization
