# 🏃‍♂️ Random Variables
A Random Variable X is a function that assigns a real number to each outcome in the sample space of a random experiment.

## Types of Random Variables
1. Discrete Random Variables
2. Continuous Random variables

## 🥇 Discrete Random Variables
A random variable that takes on a countable number of distinct values.
#### Example:
    data = [1,2,3,4,5,6,7,8,], data2 = [8,7,6,4,3,2,2,]

## 🥈 Continuous Random Variable:
A random variable that can take on any value within a given range or interval.
#### Example:
    student passing marks range = [33 - 98]
