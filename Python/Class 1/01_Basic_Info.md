# Programming
Programming means writing instructions for a computer to perform tasks using a programming language.  
A programming language helps us communicate with the computer to solve problems, create software, websites, apps, games, etc.

---------------------------------------------------------------------------------------------------------------------------------------------

## Compiler
A compiler translates the entire program at once into machine code before execution.

### Process
1. Write the program
2. Compile the program
3. If no errors → executable file is created
4. Run the program

#### Example Languages
- C
- C++
- Go

#### Example
If a C program has 10 errors, the compiler shows all errors after compilation.

### Advantages
- Program runs faster
- Errors are shown together

--------------------------------------------------------------------------------------------------------------------------------------------
 
 ## Interpreter
An interpreter translates the program line by line and executes it immediately.

### Process
1. Write the program
2. Interpreter reads one line
3. Executes that line
4. Moves to the next line

#### Example Languages
- Python
- JavaScript
- Ruby

#### Example
If there is an error on line 5, the interpreter stops there and shows the error.

### Advantages
- Easier debugging
- Good for scripting and testing
